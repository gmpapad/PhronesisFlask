JSON Schema Requirements

Your JSON file must include these required fields:

  • slug - Unique identifier (lowercase, hyphens)
  • title - Display name for the perspective
  • summary - Brief description
  • order - Numeric ordering (lower numbers appear first)
  • lessons - Array of lesson objects
